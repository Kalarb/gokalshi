# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in gokalshi, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please email **baileyfaulk2003@gmail.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation plan within 7 days.

## Scope

This policy covers the gokalshi Go SDK itself. For vulnerabilities in the Kalshi API or platform, please contact [Kalshi](https://kalshi.com) directly.
